## Key Terms Cheatsheet

| Term | One-line definition |
|------|---------------------|
| DevOps | Culture + practices + tools that unify Dev and Ops to deliver software faster and more reliably |
| SDLC | The structured phases of planning, building, testing, and releasing software |
| CI | Continuously merging and automatically testing code changes |
| CD | Automatically deploying tested code to production (or staging) |
| IaC | Managing infrastructure through version-controlled code instead of manual clicks |
| SRE | Engineering discipline that applies software engineering to operations problems |
| Shift Left | Moving testing and security earlier (left) in the development pipeline |
| Golden Path | A paved, opinionated set of tools and practices the org recommends teams use |
| DORA Metrics | 4 key metrics measuring software delivery performance (deployment frequency, lead time, MTTR, change failure rate) |
